# Clusiau-Johnson Genealogy

<img src="images/logo.png" alt="Clusiau-Johnson Genealogy logo" width="96">

GEDCOM-based family genealogy archive for the Clusiau-Johnson family.

This organization maintains a private builder repository for source GEDCOM processing and a public, redacted family tree site for family history access.

## Public family tree

* [Browse the public family tree](https://genealogy-johnson.github.io/genealogy-tree/viewer/gedcom-viewer.html)
* [View the public tree repository](https://github.com/genealogy-johnson/genealogy-tree)

## Repository structure

* `genealogy-tree` - public redacted family tree site
* `genealogy-builder` - private builder repository for GEDCOM source data, validation, redaction, metrics, and publishing

## Privacy note

The public tree is generated from private source data and is intended to exclude living/private individuals before publication.
